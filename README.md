# 2. Install/Run

LEILA is an R/Shiny application hosted on GitHub. This guide explains how to run the app locally on your personal computer.

## Requirements

Before running the application, ensure you have installed:

- [R](https://cran.r-project.org/) (latest version recommended)
- [RStudio](https://posit.co/download/rstudio-desktop/) (recommended but optional)

## Install Required R Packages

Open server.R in RStudio and click to the green button on the top right to run
